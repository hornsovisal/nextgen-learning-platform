# Module 4: Passwords, Authentication, and Access Control

## Learning Objectives

- Create strong password/passphrase policies.
- Understand multi-factor authentication (MFA).
- Apply access control best practices.

## Password Best Practices

- Use long passphrases (12+ characters).
- Avoid reused passwords across services.
- Prefer password managers for generation and storage.
- Rotate credentials immediately after suspected compromise.

## Authentication Factors

1. **Something you know** (password/PIN)
2. **Something you have** (authenticator app/token)
3. **Something you are** (biometric)

Using two or more factors significantly reduces account takeover risk.

## Access Control Concepts

- **RBAC (Role-Based Access Control):** permissions based on role.
- **Least Privilege:** minimum access required.
- **Separation of Duties:** split sensitive tasks among multiple people.
- **Periodic Access Review:** remove stale permissions.

## Common Mistakes

- Shared admin accounts
- Default passwords left unchanged
- Broad permissions given “temporarily” and never revoked

## Quick Checklist

- MFA enabled for all critical accounts
- Admin accounts separated from normal user accounts
- Failed login monitoring enabled

## Summary

Identity is the new perimeter. Strong authentication plus strict access control prevents many breaches.
