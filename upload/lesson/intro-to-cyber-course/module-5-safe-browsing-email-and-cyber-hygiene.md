# Module 5: Safe Browsing, Email, and Cyber Hygiene

## Learning Objectives

- Identify unsafe browsing and email patterns.
- Build secure daily habits (“cyber hygiene”).
- Respond safely to suspicious activity.

## Safe Browsing Habits

- Check domain spelling carefully.
- Avoid downloading software from untrusted sources.
- Verify HTTPS but remember lock icon alone is not enough.
- Keep browser and extensions updated.

## Email Safety

- Inspect sender address, not display name only.
- Hover over links before clicking.
- Treat urgency and fear language as a red flag.
- Never send passwords or OTP codes by email.

## Cyber Hygiene Routine

- Weekly updates (OS, browser, apps)
- Regular backup verification
- Password manager usage
- MFA on all important accounts
- Log out from shared/public devices

## If You Suspect Phishing

1. Do not click links or open attachments.
2. Report the message to IT/security.
3. If you already clicked, reset password immediately.
4. Revoke active sessions where possible.

## Summary

Small, consistent habits provide strong protection. Cyber hygiene is continuous, not a one-time setup.
