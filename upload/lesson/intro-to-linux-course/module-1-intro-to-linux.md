# Intro to Linux

Linux is a free and open-source operating system used in servers, cloud systems, cybersecurity labs, and developer machines.

## Why Linux Matters

- Most web servers run Linux.
- Security tools are commonly built for Linux first.
- Linux gives strong control through command-line tools.

## Key Linux Concepts

- **Kernel**: Core part of the OS that manages hardware and system resources.
- **Distribution (Distro)**: A packaged Linux system like Ubuntu, Debian, or Fedora.
- **Shell**: Interface for running commands, commonly `bash`.

## First Commands

- `pwd` shows your current directory.
- `ls` lists files and folders.
- `cd` changes directory.
- `whoami` shows the current logged-in user.

## Practice

Try these commands in order:

1. `pwd`
2. `ls`
3. `cd ..`
4. `whoami`

You are now ready to explore the Linux file system in the next module.
